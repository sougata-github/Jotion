## Key Features:

- Real-time Database
- Notion-style Editor
- Light and Dark Mode
- Infinite Children Documents
- Trash can & soft delete
- Authentication
- Real time updates
- File upload
- File deletion
- File replacement
- Expandable sidebar
- Fully responsive
- Publish your note to the web
- Cover image of each document
- Recover deleted files

## Tech-Stack:

- NextJS 14
- Tailwind CSS
- TypeScript
- Clerk
- Convex
- Shadcn UI
- Edgestore
- BlockNote Editor
- Next themes
- Sonner
- Zustand
- Zod
